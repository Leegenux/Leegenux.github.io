## 基本概念

一些基本概念、缩写的解释:
- NR：New Radio，是5G技术标准的一个代号。
- SRS：Sounding Reference Signal，信道探测参考信号，也称为导频信号。
- cyclic prefix：循环前缀，采用循环前缀有两个作用
  - 提供一个有效标识，隔开相邻两个符号，减弱码间串扰
  - 保证有时延的OFDM信号在FFT积分周期内总是具有整数倍周期

## 5G带来的改变

5G可在1GHz到100GHz工作。通常情况下，低频基站具有较大的服务覆盖范围。

为了保障服务质量，减少频谱冲突，5G基站采用授权频谱为主干、其他频谱为补充的策略进行建设。

5G的愿景是：
- eMBB：enhanced Mobile Broadband，极高的吞吐量、增强的频谱利用效率、扩大的基站覆盖面积。
- URLLC：Ultra Reliable Low Latency Communications，低延迟、超强的可靠性、精确定位
- mMTC：massive Machine Type Communications，高网络连接密度、高能量效率、低复杂度

## NR的物理层

无线通信的骨干就是物理层。NR技术的核心部件包括：调制方式、波形、帧结构、导频、多天线传输和信道编码。

### 调制方式

5G通信技术可支持4G技术支持的QPSK、16QAM、64QAM和256QAM调制。

除此之外，5G还支持了上行链路上的л/2-BPSK，以提供在低数据率条件下的高能量效率，这对于mMTC十分重要。
